Decentralized wallet for ERC20 and Ether

Key features:

  • All kinds of any token exchange
  • Storing private keys on the mobile device (trustless, security)

Extra features:

  • Confirming transactions via Ledger (Ledger blue for iOS, All Ledgers for Android)
  • Scheduled token sending at preset time and periodicity.
  • Accepting payment requests from wallet users
  • ENS Support (Ethereum Name Service, sending to address via domain name)
  • Watch Wallet - tracking account assets without inserting your private key

Decentralized ERC20 token exchange (seamless)

Key features:

  • No registrations and no KYC (permissionless)
  • Common liquidity 0x pool , based dAPPs
  • Open source

Extra features:

  • 1-click exchange
  • Fully functional exchange with classic trading instruments in a single app (graphs, order book and etc)
  • Trade orders executed straight from the app
